  • Admin Ranks

    Four types of supervision

    Admin: Blue colour can block device and IP, kick out, mute, renew mic time and mic queue options

    Superadmin : Green colour , same as admin plus : Get user info , more option for mic queue options , and unblock users and show room history

    Master : Red and Pink color , full control

    Root : Orange colour , full control plus ability to use Network program

    Room owner can set privileges for any other account and apply settings immediately

    Everybody can see his/her privileges from privileges form

    To receive camera photo right-click on the name and select view webcam option

  • Control Panel

    lametna control panel is easy to use and fully secure with many options which give room owner full control

    General :

    Contains accounts info such as: Room name, room owner, email, room capacity, online now, renew date, and account limit

    Backup and Restore :

    Now room owner can backup and restore all room data easily

    You can daily backup the room settings and accounts, blocked PC list , blocked IP list, muted list and admins report

    You can estoring the backup at any time by clicking on ( Restore )

    Manage Accounts :

    This section available only for main master account and roots and every master has permissions

    By mange accounts you can add, edit, and delete accounts and see accounts counters

    Lock machine is an option that allows you to protect any account from hacking , only one device can login using locked name

    Room Settings:

    This section available only for main master account and roots and every master has permissions

    Room Design : Changing room colors (Title, background, Frame, Borders, Font ) and Flat Frames , you can also select room style and text chat style ( new- old)

    You can also change room title and welcome message

    Voice : Select talking durations for guests, members, admins, superadmins, masters , and you can set talk permissions to : All users, members and administrators only, administrators only , and disable mic.

    Webcam: Select video quality ( Low, Medium, High) , and set webcam publish permissions to : All users, member and administrators only, administrators only , and disable webcam.

    Emoticons : Can select emoticons groups or disable all

    Private Messages : Set permissions to : All users, member and administrators only, administrators only , and disable private messages.

    Advanced Settings : You can open room for all users or lock it to be for members and administrators only or Gate ,(selecting Gate all requests need approval for login.)

    From Advanced Settings You can set all masters privileges as : Allow adding masters , and change room setting if many masters have permissions to add master accounts or change setting , you can stop all masters by selecting this option

    Also you can disable users avatars , and deactivate URLS on the text chat

  • Network Program

    In Network program you have full control to manage your chat rooms on your website with full security options.

    Login:

    To login to Network you need username , password and KEY , you can save or delete your login info on your PC , and can select program language

    Manage Rooms:

    shows rooms list and info such as: Room name , owner, email , expiry date,connected users and room size

    By selecting room you can change room info such as : Room name , email .. etc plus change main master account password and room code

    You can also set accounts limit for rooms as member, admin, superadmin, master account

    Delete all room accounts

    Change settings , and see room log and administrator report

    Unblock all pc's , IP users

    Root Accounts:

    Add , delete root account

    Add root account for one or more room under your account

    Set privilege to root to open Network program or not , lock device with name so that nobody can login to room or Network program from other device

     

    Online users:

    Shows all users online in all rooms with IP, country, nickname color,user ID, and type

    By right-clicking on the name you can kick out user , block device , block IP

    You can also broadcast messages to all online users

    Blocked users:

    Shows all blocked users devices and IP's , you can unblock any name by right click and select unblock

    Network Log:

    Shows all activity on the Network program such as : Add accounts, delete accounts , unblock user, kicked out user , broadcast message , and all actions on the Network program . no body can delete these logs .

    My account:

    Shows your account info such as: Website name, email, renew date, online users , and total accounts limit calculator
